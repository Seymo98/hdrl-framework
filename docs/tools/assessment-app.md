# HDRL Assessment App

An interactive Streamlit application for conducting HDRL assessments against the framework's 64 indicators.

---

## About the App

The **UK Health Data Assistant** is a companion tool that provides:

- Interactive assessment against all 64 HDRL indicators
- Automated scoring and readiness profile generation
- Domain-level analysis and gap identification
- Visual dashboards for assessment results
- Export of assessment reports

## Access

The app is maintained in a separate repository and deployed via Streamlit Cloud.

## How to Use

1. **Navigate** to the assessment app
2. **Select** the unit of assessment (System, Service, or Both)
3. **Score** each indicator from Level 1 to Level 5
4. **Review** the automatically generated readiness profile
5. **Export** your results for reporting and planning

## Relationship to This Site

This documentation site provides the **reference material** — the detailed indicator descriptions, maturity levels, and assessment criteria.

The Streamlit app provides the **interactive assessment tool** — the interface for conducting and recording assessments.
